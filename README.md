FLOW AI — Твой ИИ-наставник по Python

FLOW AI — это инновационная образовательная платформа, разработанная для участников конкурса **Infomatrix-Asia**. 
Проект помогает начинающим программистам изучать Python, используя возможности нейросетей не для простой выдачи ответов, а для менторства.

 Основные функции:
* **AI Mentor Mode:** ИИ анализирует код и задает наводящие вопросы, помогая ученику самому найти ошибку.
* **Solution Mode:** Мгновенное исправление ошибок и предоставление чистого, рабочего кода.
* **Прогресс-трекер:** Система отслеживает количество запросов и активность пользователя в реальном времени.
* **Безопасная авторизация:** Вход через Google Firebase.

Технологический стек
Backend: Python (Flask)
AI: Google Gemini 2.0 Flash API
Frontend: HTML5, CSS3 (Modern UI), JavaScript (ES6+)
Database/Auth: Firebase Auth
Hosting: Render

Структура проекта
app.py — Основной сервер и логика приложения.
requirements.txt — Список необходимых библиотек для развертывания.
templates/ — Шаблоны пользовательского интерфейса.
firebase.json — Конфигурация интеграции с Firebase.

Как запустить локально
1. Клонируйте репозиторий.
2. Установите зависимости: `pip install -r requirements.txt`.
3. Добавьте ваш `GOOGLE_API_KEY` в переменные окружения.
4. Запустите сервер: `python app.py`.
